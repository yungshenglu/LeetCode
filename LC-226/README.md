# LC-226 - Invert Binary Tree

Invert a binary tree.

> * Difficulty: **EASY**

---
## Examples

```
Input:
     4
   /   \
  2     7
 / \   / \
1   3 6   9
Output:
     4
   /   \
  7     2
 / \   / \
9   6 3   1
```
