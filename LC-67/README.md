# LC-67 - Add Binary

Given two binary strings, return their sum (also a binary string).

The input strings are both **non-empty** and contains only characters `1` or `0`.

> * Difficulty: **EASY**

---
## Examples

```
Input: a = "11", b = "1"
Output: "100"
```

```
Input: a = "1010", b = "1011"
Output: "10101"
```

```
Input: a = "0", b = "0"
Output: "0"
```

---
## Solutions

1. Simulation
    * Time complexity: $O(n)$
    * Space complexity: $O(1)$
