# LC-303 - Range Sum Query - Immutable

Given an integer array nums, find the sum of the elements between indices `i` and `j` (`i <= j`), inclusive.

> * Difficulty: **EASY**

---
## Examples

```
Given nums = [-2, 0, 3, -5, 2, -1]

sumRange(0, 2) -> 1
sumRange(2, 5) -> -1
sumRange(0, 5) -> -3
```

---
## Solutions

1. Dynamic Programming
    * Time complexity: $O(1)$
    * Space complexity: $O(n)$
