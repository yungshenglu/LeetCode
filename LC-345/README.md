# LC-345 - Reverse Vowels of a String

Write a function that takes a string as input and reverse only the vowels of a string.

> * Difficulty: **EASY**

---
## Examples

```
Input: "hello"
Output: "holle"
```

```
Input: "leetcode"
Output: "leotcede"
```

```
Input: " "
Output: " "
```

```
Input: "aA"
Output: "Aa"
```

```
Input: "a a"
Output: "a a"
```

---
