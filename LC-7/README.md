# LC-7 - Reverse Integer

Given a 32-bit signed integer, reverse digits of an integer.

> * Difficulty: **EASY**

---
## Examples

```
Input: 123
Output: 321
```

```
Input: -123
Output: -321
```

```
Input: 120
Output: 21
```

---
## Solutions

1. Math Approach
    * Time complexity: $O(\log_{10}{n})$
    * Space complexity: $O(1)$
