# LC-326 - Power of Three

Given an integer, write a function to determine if it is a power of three.

> * Difficulty: **EASY**

---
## Examples

```
Input: 27
Output: true
```

```
Input: 0
Output: false
```

```
Input: 9
Output: true
```

```
Input: 45
Output: false
```

---
## Follow Up

* Could you do it without using any loop/recursion?

---
## Solutions

1. Loop Approach
    * Time complexity: $O(\log_{3}n)$
    * Space complexity: $O(1)$
2. Base Changing
    * Time complexity: $O(1)$
    * Space complexity: $O(1)$
