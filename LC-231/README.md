# LC-231 - Power of Two

Given an integer, write a function to determine if it is a power of two.

> * Difficulty: **EASY**

---
## Exmaples

```
Input: 1
Output: true
Explanation: 20 = 1
```

```
Input: 16
Output: true
Explanation: 24 = 16
```

```
Input: 218
Output: false
```

```
Input: 0
Output: false
```
