# LC-83 - Remove Duplicates from Sorted List

Given a sorted linked list, delete all duplicates such that each element appear only once.

> * Difficulty: **EASY**

---
## Examples

```
Input: 1->1->2
Output: 1->2
```

```
Input: 1->1->2->3->3
Output: 1->2->3
```

```
Input: 1->1->1
Output: 1
```

```
Input: NULL
Output: NULL
```

---
## Solutions

1. Naive Approach
    * Time complexity: $O(n)$
    * Space complexity: $O(1)$
