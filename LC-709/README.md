# LC-709 - To Lower Case

Implement function `ToLowerCase()` that has a string parameter `str`, and returns the same string in lowercase.

> * Difficulty: **EASY**

---
## Examples

```
Input: "Hello"
Output: "hello"
```

```
Input: "here"
Output: "here"
```

```
Input: "LOVELY"
Output: "lovely"
```

```
Input: "Alph&"
Output: "alph&"
```

---
## Solutions

1. Naive Method
    * Time complexity: $O(n)$
    * Space complexity: $O(1)$
