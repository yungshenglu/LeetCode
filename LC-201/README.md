# LC-201 - Bitwise AND of Numbers Range

Given a range `[m, n]` where `0 <= m <= n <= 2147483647`, return the bitwise `AND` of all numbers in this range, inclusive.

> * Difficulty: **MEDIUM**

---
## Exmaples

```
Input: [5,7]
Output: 4
```

```
Input: [0,1]
Output: 0
```

---
## Solutions

1. Shift Operations
    * Time complexity: $O(n)$
    * Space complexity: $O(1)$
