# LC-203 - Remove Linked List Elements

Remove all elements from a linked list of integers that have value val.

> * Difficulty: **EASY**

---
## Examples

```
Input:  1->2->6->3->4->5->6, val = 6
Output: 1->2->3->4->5
```

---
## Solutions

1. Naive Approach
    * Time complexity: $O(n)$
    * Space complexity: $O(1)$
